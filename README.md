# Databricks Data Engineering — Mastery Portfolio

A 16-week structured learning journey from Spark fundamentals to 
Solution Architect level, built around 3 core textbooks and hands-on 
enterprise-grade notebooks.

## Stack
- Apache Spark 3.5.3 (local) + Databricks Free Edition (cloud)
- Delta Lake 3.2.0
- Python 3.11

## Structure
| Folder | Phase | Weeks |
|---|---|---|
| 01-foundations | Spark Core, Delta Lake basics | 1–4 |
| 02-core-engineering | Ingestion, ETL, Quality, Orchestration | 5–9 |
| 03-advanced | Medallion, Streaming, Security, Performance | 10–13 |
| 04-capstone | End-to-end enterprise pipeline | 14–16 |
